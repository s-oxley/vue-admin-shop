# vue-admin-shop

This template should help get you started developing with Vue 3 in Vite.

## Pasos para Dev

1. Clonar el repo
2. Instalar las dependencias
3. Crear el archivo .env basado en el archivo .env.template
4. Ejecutar el proyecto ```npm run dev````
